If you want to make splash screen use
MzHelper.setTimer(Activity activity,
int milliseconds, TimerInterface timerInterface);

Example Code: 

	  
public class SplashActivity extends Activity
 implements TimerInterface {
		
@Override
public void startTimer(){
Intent intent = new Intent(getApplicationContext(), MainActivity.class);
startActivity(intent);
}

@Override
protected void onCreate(Bundle bundle){
super.onCreate(bundle);
setContentView(R.id.activity_splash);
MzHelper.setTimer(this, 500, this);
}
}

If you want to get Apk File for any app use MzHelper.getApkFile(Context context, String packageName);

Example Code: 

	  

public class MainActivity extends Activity {


@Override
protected void onCreate(Bundle bundle){
super.onCreate(bundle);
setContentView(R.id.activity_main);
File f = MzHelper.getApkFile(this, "com.google.android.gms");
try {
Intent share = new Intent();
share.setAction(Intent.ACTION_SEND);
share.setType("application/vnd.android.package-archive");
share.putExtra(Intent.EXTRA_STREAM, Uri.fromFile(f));
startActivity(Intent.createChooser(share, "Choose App"));
}
catch (PackageManager.NameNotFoundException pm){

}
}
