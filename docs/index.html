If you want to make splash screen use MzUtils.setTimer(Activity activity, int milliseconds, TimerInterface timerInterface);

Example Code: 

	  
public class SplashActivity extends Activity implements TimerInterface {
		
@Override
public void startTimer(){
Intent intent = new Intent(getApplicationContext(), MainActivity.class);
startActivity(intent);
}

@Override
protected void onCreate(Bundle bundle){
super.onCreate(bundle);
setContentView(R.id.activity_splash);
MzUtils.setTimer(this, 500, this);
}
}

If you want to make app do thing every 500 ms, 1s or 1m .. etc use MzUtils.setTimerForEver(Activity activity, int milliseconds, TimerInterface timerInterface);

Example Code: 

	  
public class MainActivity extends Activity implements TimerInterface {

@Override
public void startTimer(){
Log.d("MainActivity", "Done");
		}

@Override
protected void onCreate(Bundle bundle){
super.onCreate(bundle);
setContentView(R.id.activity_main);
MzUtils.setTimerForEver(this, 1000, this);
}
}

If you want to make app play music use startMedia(Context context, int mediaId);

Example Code: 

	  
public class MainActivity extends Activity {


@Override
protected void onCreate(Bundle bundle){
super.onCreate(bundle);
setContentView(R.id.activity_main);
MzUtils.startMedia(this, R.raw.song);
// If you want stop use stopMedia();
}

}

If you want to get Location Y for View use MzUtils.getLocationY(android.view.View view);

Example Code: 

	  
public class MainActivity extends Activity {


@Override
protected void onCreate(Bundle bundle){
super.onCreate(bundle);
setContentView(R.id.activity_main);
LinearLayout linear = findViewById(R.id.mContent);
int locY = MzUtils.getLocationY(linear);
Log.d("MainActivity", /*To convert int to String*/ MzUtils.getString(this, locY));
}
}

If you want to get Drawable in old version of Android use MzUtils.getDrawable(Context context, int resId);

Example Code: 

	  
public class MainActivity extends Activity {


@Override
protected void onCreate(Bundle bundle){
super.onCreate(bundle);
setContentView(R.id.activity_main);
ImageView iv = findViewById(R.id.myIv);
Drawable dr = MzUtils.getDrawable(this, R.drawable.mDr);
iv.setImageDrawable(dr);
}
}

public class MainActivity extends Activity {
